# GitHub Configuration

This directory contains GitHub-specific configuration files.

## Workflows

The `workflows/` directory contains CI/CD workflow templates that are **disabled by default**.

⚠️ **Important**: To prevent accidental GitHub Actions costs, workflow files must be explicitly activated by renaming from `.example` to `.yml`.

See the main [README.md](../README.md#-cicd-optional-disabled-by-default) for instructions on enabling/disabling CI/CD.
