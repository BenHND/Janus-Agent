## 🎯 What is Janus?

Janus is your **personal, local, and intelligent AI voice assistant** that runs 100% on your computer.

### Key Features

- 🔒 **Total Privacy** - No cloud, all processing local
- ⚡ **Lightning Fast** - Optimized for Apple Silicon & Windows
- 🧠 **True Intelligence** - Understands context, sees screen, learns from you
- 👁️ **Computer Vision** - Works with any application
- 🔄 **Multi-Step Automation** - Complex workflows across apps

---

## 🚀 Quick Start

1. **Install** Janus (see Installation section in manual)
2. **Grant permissions** (Microphone, Accessibility, Screen Recording)
3. **Calibrate** your microphone (5 phrases)
4. **Try your first command:** *"Open Safari and go to YouTube"*
